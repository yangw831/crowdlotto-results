# CrowdLotto Results

This repository is intended to be the public GitHub Pages repository for CrowdLotto static result files.

It should publish only the mobile app's public data files under:

* `drawing_results/mm/*.html`
* `drawing_results/pb/*.html`
* `drawing_results/estimated_jackpot.html`
* `drawing_results/pb_estimated_jackpot.html`

The mobile app expects these public paths:

* `/drawing_results/mm/YYYY-MM-DD.html`
* `/drawing_results/pb/YYYY-MM-DD.html`
* `/drawing_results/estimated_jackpot.html`
* `/drawing_results/pb_estimated_jackpot.html`

This repo should not contain scraper code, secrets, or private automation logic.
